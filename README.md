# Euler–Bernoulli Beam Model for Flapping Wings

This repository provides numerical implementations of the Euler–Bernoulli beam theory
applied to bioinspired flapping-wing structures.

The model includes time-domain integration, kinematic excitation at the root,
and reconstruction of the deformed wing geometry.
